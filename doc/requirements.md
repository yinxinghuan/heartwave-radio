# Requirements

## 1. Overview

《心动电台 / Heartwave》是一款 60 秒社交默契游戏：玩家连续完成 5 轮“此刻更想要哪一种感觉”的双选题，累积频道频率与连击分数，生成一张可发布到社区频率墙的个性电台卡。

## 2. Visual Design

- 画布采用 390 × 760 px 手机竖屏，背景为午夜蓝 `#090B18`，叠加玫红 `#FF4F8B`、电光紫 `#8B5CF6` 与薄荷青 `#66F2D5` 的柔光渐变；四周保留 20 px 安全边距。
- 整体风格为“AlterU 床头实体 FM”：品牌主色采用油墨粉 `#E83F78`、薄荷绿 `#69CDB6`、机身紫 `#21152B`、标签纸 `#F4E7D3` 与墨黑 `#1B1320`；取消液态玻璃和大面积发光，改用 2–3 px 实线、5–8 px 硬偏移阴影、纸张标签、塑料机身与扬声器织网材质。
- 开始页顶部为印刷式 AlterU `A` 台标和电台呼号，中部为 200 px 扬声器织网与固定调频刻度，标题使用本地 Onest 900、54 px/0.82 行高与 `-0.075em` 字距；主按钮高 58 px。界面根据系统语言自动切换中文 / 英文，并允许 `game_locale=zh|en` 手动覆盖。
- 游戏页顶部显示 5 段机械信号格，每段高 7 px、间距 4 px；题目卡位于中央，两个纸质选项卡各高 204 px、7 px 圆角，使用奶油纸与 AlterU 粉双色丝网印刷，包含统一的 38 px 线性 SVG 信号图标、20 px 标题和 11 px 描述。
- 每次选择后出现 700 ms 的频率闪光、`+N` 浮分与 900 ms 台词气泡；连击达到 2 时在顶部显示 20 px 的 `×N STREAK` 发光标签。
- 结果卡宽 330 px、高 440 px，包含渐变唱片、频道名称、频率数值、三枚个性标签与 5 格选择轨迹；社区墙使用单列可滚动卡片，每张卡最小高 176 px。
- 动画：唱片 12 s 线性旋转，声波 1.6 s 循环，页面入场 420 ms，卡片按下 100 ms，结果揭晓 900 ms；`prefers-reduced-motion` 下关闭循环旋转与大位移动画。
- 素材清单：Aigram 半透明水印 SVG；其余唱片、波形、星点、频道卡与图标均由 CSS 和 Unicode 符号绘制，不依赖外部图片。

## 3. Game Mechanics

- 每局固定 5 轮，初始剩余时间 60 秒、分数 0、连击 0；开始后前 1.5 秒为缓冲期，计时正常但不触发超时失败。
- 每轮从 10 组题库中按局随机种子抽取一题且不重复；左右选项分别带有 `warm`、`wild`、`quiet`、`bright` 四维频率权重。
- 选择基础得分为 100；在 4 秒内选择额外获得 `max(0, round((4000 - 用时毫秒) / 40))`，连续选择同一频率主维度时连击加 1，否则重置为 1；连击奖励为 `20 × (combo - 1)`。
- 每次选择产生 8 个粒子、一个浮动分数与一句对应频道台词；完成第 5 轮后，将四维累计最高项映射为频道，频率为 `88.0 + (总分 mod 200) / 10` MHz。
- 频道包括四套基础视听气质：暖夜台使用 70 BPM Rhodes 与珊瑚粉；失重台使用 96 BPM 断奏方波与酸性紫；月背台使用 60 BPM 稀疏钟声与冰蓝；晨光台使用 108 BPM 明亮拨弦与日光黄。结果内容由主频道 × 副频道生成 12 种复合人格，每种包含独立标题、描述、3 条午夜广播和标签组。作品共享可扩展的“广播档案框架”：刊号、频率、标题、标签和五段选择轨迹是稳定锚点，中央主视觉是可替换画面槽。第一层受控视觉 DNA 包含等高接收域、折叠频谱带、字符坐标阵列 3 种主画面语法；第二层包含疏朗与密集 2 种信息密度；第三层包含上行与下行 2 种信号方向，共 12 种受控组合。作品 ID 决定组合且发布后保持稳定；频道只改变纸张和强调色。
- 本地最高分以 `heartwave_radio_best` 保存。玩家可无限重新调频，但每个本地自然日只允许公开播出 1 张卡；每日锁只读取云存档镜像中的 `lastBroadcastDay`，不读取平台聚合统计。个人 `broadcasts` 最多保存最近 20 条，社区墙展示合并后最新 24 条，但数据层保留每个用户的完整存档数组。本人可在完整广播详情中两次确认删除旧作品；删除不返还当天公开播出机会。

## 4. Controls

- 开始、选项、再来一次、返回首页、发布按钮使用 `onPointerDown`；按下即触发并屏蔽浏览器点击高亮。
- 键盘支持 `1` / `←` 选择左项，`2` / `→` 选择右项，`Enter` 在开始页开始或在结果页再来一次。
- 社区墙内卡片与作者头像姓名使用 `onClick`，以便触摸滑动时不误触；作者头像姓名点击后在 Aigram 内打开对应用户主页。
- 社区墙使用垂直触摸滚动，`touch-action: pan-y`；墙卡使用 `onClick` 打开完整广播详情，作者头像姓名独立点击打开用户主页，关闭与删除按钮也使用 `onClick`；底部固定返回按钮不参与滚动。

## 5. Win / Lose Conditions

- 完成 5 轮即胜利并进入结果页，展示最终得分、历史最高、频道名称、MHz 频率、性格标签、选择轨迹、发布到频率墙、查看频率墙、再来一次与返回首页。
- 60 秒耗尽则提前结束，已完成至少 1 轮时仍生成频道卡；0 轮时显示“信号丢失”结果与 0 分。
- 社区墙为空时显示发光空状态与“成为第一个开台的人”；非 Aigram 环境仍可本地发布并查看自己的频道卡。

## 6. Sound Effects

- 首次交互恢复 Web Audio；所有音效由振荡器合成，主音量不超过 0.09。
- 按钮：600 Hz 到 420 Hz 的 sine，时长 45 ms，音量 0.035。
- 开始：330 Hz 与 494 Hz 两个 sine 依次响起，各 120 ms，间隔 70 ms，音量 0.055。
- 选择：基础反馈使用 480 Hz 与 760 Hz triangle，时长 90 ms；随后按选择维度叠加频道乐句：warm 为 261.6/329.6/392 Hz 正弦和弦，wild 为 220/330/494/370 Hz 方波断奏，quiet 为 523.3/784 Hz 长尾钟声，bright 为 392/493.9/587.3/784 Hz 三角波上行。
- 连击：combo ≥ 2 时增加 660 Hz 到 990 Hz 的 sine 滑音，时长 150 ms，音量 0.05。
- 完成：392、494、587、784 Hz 四音上行，每音 180 ms、间隔 90 ms，音量 0.06。
- 超时：180 Hz 到 90 Hz 的 sawtooth，时长 320 ms，音量 0.035。
- 发布成功：740 Hz 与 1110 Hz 两个 sine，分别 110 ms 和 220 ms，间隔 80 ms，音量 0.045。
